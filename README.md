## About

Super Nintendo emulator for classic 16-bit games.

## Features

- ROM support
- Save states
- Controller mapping
- Graphics filters

## Installation

### From App Store
Open the **App Store** in zOS and search for "SNES".

### Via npm
```bash
npm install @zos-apps/snes
```

## Usage

- Load ROM to play
- Configure controls
- Use save states

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Start |
| `Shift` | Select |
| `WASD` | D-pad |
| `IJKL` | Buttons |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
